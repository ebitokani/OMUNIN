ログイン・招待

ログイン方法

1
招待リンクを受け取る
管理者から送られてくる招待リンクをタップします。招待リンクは7日間有効・1回のみ使用可能です。
2
Googleでログイン
「Googleでログイン」ボタンを押して、Googleアカウントでログインします。
3
アプリを使い始める
ログイン後は次回から招待リンクなしで直接ログインできます。
⚠️ 注意

招待リンクなしでログインしようとすると「アクセス権限がありません」と表示されます。管理者に招待リンクの発行を依頼してください。

案件登録・管理

新規案件の登録

画面下の「➕ 新規登録」タブから登録します。

項目内容
案件ステータス見込み / 確定 / 完了 から選択
担当営業担当者名を入力
ビルダー名施工会社名を入力（必須）
工事名〇〇様邸など（必須）
施主名施主のお名前
住所現場住所（入力するとマップ表示）
工種柱状 / 鋼管 / 表層 / ピュアパイル から選択

案件一覧・検索

「📋 案件一覧」タブに全案件が表示されます。左にスワイプすると削除ボタンが表示されます。

案件の編集

案件詳細画面で「✏️ 編集」ボタンから基本情報を修正できます。

💡 ポイント

住所を入力するとGoogleマップへのリンクが自動表示されます。

🚛 大型車ルート

案件詳細の「現場の場所」カードに「🚛 大型車ルートを開く」ボタンが表示されます（住所登録済みの場合）。タップするとGoogleマップのナビが起動します。スマホではGoogleマップアプリが自動で開きます。

現地下見の記録

案件詳細の「STEP2 現地下見」から記録します。

項目内容
下見日現地調査を行った日付
下見担当担当者名
地形平坦 / 傾斜 / 谷 / その他
地質乾燥/軟らかい / 乾燥/硬い / 湿潤/軟らかい / 湿潤/硬い
植生草 / 低木 / 高木 / なし
空中障害有 / 無
高低差有 / 無
道路幅実測値（m）を入力
搬入障害 / 重機自走 / 大型車搬入有 / 無 / 可 / 不可 から選択
水道 / トイレ有 / 無

施工情報の入力・編集

案件詳細の「STEP3 施工情報」から入力、または詳細画面の「✏️ 編集」ボタンから後から修正できます。

項目内容
杭径 (φ)数値入力（例：300）
杭長 (m)数値入力（例：8.0）
本数数値入力（例：20）
掘削深度 (m)数値入力
改良厚 (m)数値入力
設計GLテキスト入力（例：GL=KBM+0.5）
盛土予定有 / 無
盛土量 (mm)数値入力
有機質土・黒ボク有 / 無
設計接地圧 (KN/㎡)数値入力
💡 ポイント

施工情報は詳細画面の「✏️ 編集」ボタンからいつでも修正できます。未入力の場合も「未入力」と表示されて編集ボタンから入力できます。

写真管理・撮影

写真の追加

1
案件詳細の写真セクションを開く
「基本写真」または「特記写真」セクションの各項目に📷ボタンがあります。
2
写真を選択する
カメラで撮影またはライブラリから選択します。
3
自動保存される
写真には撮影日時が左下に自動で入ります。同じ項目に再度追加すると古い写真は自動削除されます（上書き保存）。

基本写真の項目

全景①②・前面道路・搬入口・地縄・丁張・境界杭・既存建物・外周擁壁・水道・メーターなど（設定から変更可能）

特記写真

項目名を自由に入力して追加できます。よく使う項目名は設定からテンプレート登録できます。

⚠️ 上書き保存について

同じ項目に写真を追加すると前の写真が削除されます。複数枚残したい場合は項目名を変えて登録してください。

電子小黒板モード
NEW

写真項目の「📋 小黒板」ボタンから起動します。案件情報を小黒板として画面に重ねながら撮影できます。

調査モード（デフォルト）

工事名・施主名・工種・撮影日時を表示します。

施工モード

小黒板右上の「施工」ボタンに切り替えると、杭径・杭長・本数が追加表示されます。

メモ機能

画面下のテキストエリアにメモを入力すると小黒板にリアルタイムで反映されます。

1
「📋 小黒板」ボタンをタップ
カメラが起動し、左下に緑の小黒板が表示されます。
2
モードを選択（調査 / 施工）
右上のボタンで切り替えます。
3
シャッターボタンで撮影
小黒板が合成された状態で自動保存されます。
📊
Excel出力
NEW

案件一覧画面の「📊 Excel出力」ボタンから全データをExcelファイルに出力できます。

出力内容

シート名内容
案件マスタ全案件の基本情報
現地下見全案件の現地下見情報
施工情報全案件の施工情報

ファイル名は OMNIN_YYYYMMDD.xlsx の形式で保存されます。

設定

基本写真項目の編集

写真管理の「基本写真」に表示される項目をカスタマイズできます。追加・削除・並べ替えが可能です。

特記テンプレートの登録

特記写真追加時の候補として表示されるテンプレート項目を登録できます。よく使う項目名を登録しておくと便利です。

管理者機能
管理者のみ

管理者権限を持つユーザーのみ設定タブに表示されます。

招待リンクの発行

1
設定タブ →「🔗 招待リンク発行」
「➕ 招待リンクを生成」ボタンをタップします。
2
URLをコピーして共有
「📋 コピー」ボタンでURLをコピーし、LINEやメールで送ります。
💡 招待リンクについて

招待リンクは7日間有効・1回のみ使用可能です。使用済みのリンクは再利用できません。

ユーザー管理

登録済みユーザーの一覧表示と、管理者 ↔ メンバーのロール変更ができます。

ロールできること
管理者全機能 + 招待リンク発行 + ユーザー管理
メンバー案件登録・編集・写真管理・Excel出力
